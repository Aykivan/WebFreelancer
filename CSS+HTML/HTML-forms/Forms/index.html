Теория

Тег form используется для создания HTML-формы, в которой пользователь может ввести данные. Этот тег определяет область, в которой содержится контент формы, а также указывает на сервер, куда отправлять данные формы.

Атрибуты тега:

  • action="#" вставляем ссылку куда будут отправлены данные
  • method="post/get" - get - по умолчанию, post - использует шифрование и необходим для отправки файлов
  • enctype="multipart/form-data" - чтото про кодировку на сороне пользователя

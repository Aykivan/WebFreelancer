transform - применяется только к блочным обьектам, если нужно применить его к строчному тегу, неоюходибо добавить параметр display
	block или inline-block. Обьекты не влияют на окружение и к ним можно применять анимации.

translate(x, y) - сдвигает элемент на новое место, перемещая относительно изначального положения объекта вправо и вниз, 
	используя координаты X и Y. Записывается следующим образом translate (x,y);

scale(x, y) - масштабирует элементы, делая их больше или меньше. По умолчанию значения указаны как единица (1) - это реальный размер объекта. 
	Первое значение масштабирует элемент по ширине, второе — по высоте. 
	Отрицательные значения отображают элемент зеркально.

rotate(угол+deg)- поворачивает элементы на заданное количество градусов, отрицательные значения поворачивают элемент против часовой стрелки, 
	положительные — по часовой стрелке. Записываются значения как градус поворота и deg

skew(x-угол+deg, y-угол+deg) - деформирует (искажает) стороны объекта относительно координатных осей. Если указано одно значение, второе 
	будет определено браузером автоматически. Записываются значения как градус поворота и deg

matrix(a, c, b, d, x, y) - позволяет объединить несколько функций 2D-трансформаций в одной. В качестве трансформации допустимы 
	поворот, масштабирование, наклон (деформация) и изменение положения. Свойства указываются без единиц измерения.

transform-origin - позволяет сместить центр трансформации, относительно которого происходит изменение положения/размера/формы элемента
perspective, perspective-origin - установка глубины и смена точки начала координат

translate3D(x, y, z) - по аналогии со своим 2D братом это значение задает перемещение элемента, но, в 3D-пространстве.
	Помимо оси X и Y добавляется ось Z.

scale3d(x, y, z) - задает трехмерное масштабирование по одной из осей X, Y, Z

rotate3d (x, y, z, deg) - вращает элемент относительно трех осей. Элемент поворачивается под углом, задаваемым последним параметром
	относительно вектора направления X, Y, Z

matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) - задает трехмерное преобразование как однородную матрицу размером 4×4 с шестнадцатью
	значениями в столбцах

transform-style - задает стиль трансформации

Backface-visibility - cвойство определяет, будет ли видна пользователю обратная сторона элемента.


matrix3d - задает трехмерное преобразование как однородную матрицу размером 4×4 с шестнадцатью значениями в столбцах
